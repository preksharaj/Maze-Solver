# Maze-Solver
maze.py has the code to create and display the maze and find the solution to reach from the desk to the coffee machine.
the office is represented as a martix maze with " " indicating a space and # indicating a wall/desk.
The desk or Start point is represented as D and the coffee machine or end point is represented as C.

Input
-----
mazefile.txt is the input file taken to construct the office maze.
You can specify the required maze in the text file. I have just used one example maze.

Instructions to run the code
----------------------------
# python maze.py
